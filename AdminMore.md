# 🚀 관리자 페이지(Admin) 추가 기능 설계 제안

현재 구현된 예약/서명 관리 기능을 넘어, 운영 효율성과 데이터 분석 기능을 강화하기 위한 추가 기능 설계안입니다.

---

## 1. 📊 통합 통계 대시보드 (Dashboard Insights)
운영 현황을 한눈에 파악할 수 있는 시각화 지표를 제공합니다.
- **주요 지표 (KPI):**
    - 날짜별/시간별 예약 현황 추이 그래프.
    - 프로젝트별(아이디어 공모, 소상공인 협력 등) 점유율 차트.
    - 사용자 국적(언어 설정 기준) 분포도.
- **AI 분석 지표:**
    - OCR 인식 성공률 및 평균 신뢰도(Confidence) 추이.
    - 이름 번역(Transliteration) 사용 빈도 분석.

## 2. ✍️ AI OCR 검수 및 교정 도구 (OCR Review Tool)
AI가 인식한 서명 및 이름 데이터를 관리자가 최종 확인하고 수정하는 기능입니다.
- **저신뢰도 알림:** 신뢰 점수가 80점 미만인 데이터에 별도 강조 표시.
    - 관리자가 원본 서명 이미지와 인식된 텍스트를 나란히 비교(Side-by-Side).
    - 잘못 인식된 이름이나 언어를 즉시 수정 및 승인.
- **번역 이력 관리:** 수동으로 수정한 이름 데이터를 학습 데이터로 활용하거나 별도 사전(Dictionary)으로 관리.

## 3. 👥 사용자 및 권한 관리 (User Management)
서비스 이용자 및 관리자 계정을 효율적으로 관리합니다.
- **회원 목록 조회:** 가입 경로(소셜/이메일), 가입일, 최근 접속일 필터링.
- **권한 할당:** 일반 사용자, 중간 관리자(조회 전용), 시스템 관리자(모든 권한) 등 역할 기반 접근 제어(RBAC) UI.
- **접속 로그:** 관리자의 중요 데이터(예약 삭제, 개인정보 열람 등) 변경 이력 추적(Audit Log).

## 4. ⚙️ 실시간 운영 설정 (Service Configuration)
코드 수정 없이 운영 정책을 즉시 반영합니다.
- **정원 관리:** 프로젝트별/시간대별 예약 가능 인원(Capacity) 실시간 수정.
- **공지사항 관리:** 메인 화면 및 각 서비스 하단에 표시될 배너/공지 문구 편집.
- **시스템 점검 모드:** 특정 서비스 또는 전체 시스템을 점검 중으로 전환하는 토글 스위치.

## 5. 📂 데이터 엑보트 및 보고서 (Export & Reporting)
외부 활용을 위한 데이터 추출 기능을 제공합니다.
- **엑셀/CSV 다운로드:** 예약 목록, 서명 목록을 필터링된 상태 그대로 엑셀 파일로 추출.
- **증명서 일괄 생성:** 특정 기간의 서명 완료자들을 대상으로 수료증(PDF)을 일괄 생성 및 압축 파일로 다운로드.

## 6. 🌍 실시간 다국어 편집기 (I18n Live Editor)
- **번역 관리 UI:** `i18n.js` 파일을 직접 수정하지 않고, 관리자 페이지에서 각 언어별 문구를 수정하고 즉시 적용.
- **신규 언어 확장:** 신규 언어(예: 프랑스어, 독일어 등) 추가 시 템플릿을 제공하여 쉬운 로컬라이징 지원.

---

## 💡 우선순위 제안
1.  **Level 1 (필수):** 데이터 엑셀 다운로드 (정산 및 보고용)
2.  **Level 2 (운영):** 실시간 정원 관리 및 OCR 검수 도구 (운영 품질 향상)
3.  **Level 3 (분석):** 통계 대시보드 (향후 축제 기획 데이터 활용)
